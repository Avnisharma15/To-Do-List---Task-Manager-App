# To-Do List – Simple Task Manager

A clean and responsive To-Do List app built using **HTML, CSS, and JavaScript**.  
Users can add and delete tasks using a minimal and user-friendly interface.

## Features
- Add new tasks  
- Delete tasks  
- Clean and simple UI  
- Fully responsive layout  
- Beginner-friendly JavaScript

## Technologies Used
- HTML  
- CSS  
- JavaScript
